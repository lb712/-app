## 项目如何运行

```
git clone https://github.com/SmallRuralDog/lyf-vue.git
cd lyf-vue
npm install
npm run dev
```
由于开源代码使用的接口是我这边的测试接口，如果需要加入自己的业务请自行开发接口，接口可具体查看源码，或可参考`shopnc`，商城后端是查考`shopnc`开发的，接口返回字段基本和`shopnc`相同
